# BlitzCraft Launcher

Repo public du launcher BlitzCraft.

Par choix, le **code source officiel** du launcher n'est pas partagé ici (ni ailleurs en open source). Ce dépôt sert de page d'accueil GitHub : liens officiels uniquement.

## Télécharger

[Page de téléchargement officielle](https://novaclient.fr/blitzcraft/downloads)

## Transparence (auth Microsoft)

On ne reçoit jamais ton mot de passe Microsoft.

Le dépôt auth ne montre pas juste un fichier isolé : il montre la **chaîne complète** (UI -> preload -> IPC -> OAuth Microsoft -> launch).

[BlitzCraft-Launcher-Auth](https://github.com/KendiiiX/BlitzCraft-Launcher-Auth)  
Architecture : [ARCHITECTURE.md](https://github.com/KendiiiX/BlitzCraft-Launcher-Auth/blob/master/docs/ARCHITECTURE.md)

## Ce que ce repo contient

- Ce README
- Les liens officiels ci-dessus

## Ce que ce repo ne contient pas

- Le code source du launcher
- Les builds / sources internes
- Les secrets / configs serveur

Pour jouer : passe par le site de download. Pour vérifier l'auth : passe par le repo Auth.
